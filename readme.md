## 🚀 Getting Started

Each project is self-contained. To run any project locally:

```bash
# Clone the repository
git clone https://github.com/ranichandnirani/Web-projects-by-level.git

# Navigate to a project folder
cd Web-projects-by-level/0001_To-Do\ List

# Open index.html in your browser
open index.html
```

No build tools or dependencies required — just open `index.html` in any modern browser.
