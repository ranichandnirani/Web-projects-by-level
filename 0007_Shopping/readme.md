# Trendova 🛍️

Trendova is a modern and responsive shopping website built using **HTML, CSS, and JavaScript**.
This project focuses on creating a clean UI, smooth user experience, and interactive frontend features for an online fashion store.

## ✨ Features

* Responsive design for all devices
* Modern shopping website UI
* Interactive product sections
* Smooth navigation
* Clean and organized layout
* Stylish hero section
* JavaScript-powered interactions
